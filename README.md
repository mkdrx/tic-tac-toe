- Clone the project

```bash
  git clone https://github.com/mkdrx/tic-tac-toe.git
```

- Go to the project directory

```bash
  cd tic-tac-toe
```

- Install dependencies

```bash
  npm install
```

- Run dev environment

```bash
  npm run dev
```
